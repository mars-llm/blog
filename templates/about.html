About

This blog is a retro-pixel logbook for Bitcoin & Mining. Target audience: techies, but also normies who want to understand what's really happening in the mining world.

How to publish content

  • Create a new Markdown file in content/posts/.
  • Set YAML front matter (title, date, tags, level).
  • Push to main → GitHub Actions builds and deploys to Pages.

Customize the theme

  • Colors + image paths: site.yml
  • CSS: assets/css/main.css
  • Graphics: assets/img/
